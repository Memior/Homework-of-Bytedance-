# 电商平台商品列表页

> document:markdown by Typora，homework：电商

一个使用 Vue 3 + Element Plus + Pinia 构建的现代化电商商品列表页面。

## 功能特性

### 核心功能
- **商品筛选**：支持分类、品牌、价格区间、关键词多维度筛选
- **商品排序**：综合排序、价格排序、销量排序、评分排序、最新排序
- **分页加载**：支持自定义每页显示数量，智能分页导航
- **状态管理**：使用 Pinia 进行统一状态管理，支持数据缓存
- **购物车功能**：添加购物车、数量管理、本地持久化存储

### 用户体验优化
- **响应式设计**：完美适配桌面、平板、手机等多种设备
- **性能优化**：
  - 图片懒加载减少初始加载时间
  - 数据缓存避免重复请求
  - 代码分割优化打包体积
  - 防抖处理优化筛选性能
-  **流畅动画**：页面切换、商品卡片、购物车等多处动画效果
-  **交互优化**：
  - 吸顶筛选栏便于快速筛选
  - 回到顶部按钮优化长页面浏览
  - Skeleton 加载占位提升感知性能
  - Toast 提示增强操作反馈

### 技术亮点
-  Vue 3 Composition API
-  Pinia 状态管理
-  Element Plus UI 组件库
-  Vite 构建工具
-  移动端优先的响应式设计
-  LocalStorage 数据持久化

##  技术栈

- **框架**：Vue 3.3+
- **状态管理**：Pinia 2.1+
- **UI 组件**：Element Plus 2.4+
- **路由**：Vue Router 4.2+
- **构建工具**：Vite 5.0+
- **样式**：SCSS
- **HTTP 客户端**：Axios 1.6+

## 🎯 核心实现

// 主要功能

#### cartStore - 购物车状态管理
```javascript
// 主要功能
- 购物车商品管理
- 添加/删除/更新商品
- 购物车数量和总价计算
- LocalStorage 持久化
```

### 2. 性能优化措施

#### 图片懒加载，==但是一些图片可能会访问失败，官方高清数据有一些限制，所以简单的进行了替代。==
- 使用 Element Plus 的 `lazy` 属性
- 减少初始加载时间
- 优化首屏性能

#### 数据缓存
- Pinia Store 中实现缓存机制
- 避免重复 API 请求
- 提升用户体验

#### 防抖处理
- 筛选条件变化时使用防抖
- 减少不必要的计算
- 提升交互流畅度

#### 代码分割
- 路由级别的懒加载
- 第三方库分离打包
- 优化构建产物体积
